# Bloom Brew Café Menu Design

## 📌 Project Overview
This project showcases a modern and elegant café menu design created for a fictional coffee shop called **Bloom Brew Café**. The goal of this design is to create a visually appealing and easy-to-read menu that reflects the warm and welcoming atmosphere of a café.

The design uses a minimal layout, soft earthy colors, and clean typography to ensure a professional and aesthetic presentation suitable for cafés, coffee shops, and small restaurants.

## 🎨 Design Features
- Minimal and elegant café theme
- Organized menu sections
- Soft earthy color palette (cream, brown, and green)
- Decorative botanical elements
- Coffee-themed icons
- Clear typography for better readability
